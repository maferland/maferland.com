# Lessons

- When told a handoff branch contains an artifact, fetch and inspect the artifact itself before implementing or deploying. Do not substitute README/source interpretation for the requested handoff.
- For production deploy mistakes, restore the previous alias first, then continue investigation.
- Watch CSS specificity on scoped landing pages. Generic selectors like `.page a` can override button text colors and make CTAs unreadable on mobile.
- Mobile demos cannot hide the only changing part of the example. Let URL demos wrap on narrow screens so stripped params remain visible.
